---
title: 'evaluate_stability: assess stability pass/fail'
description: 'Compare diff_hom_stab against the basic and expanded ISO 13528:2022 stability criteria and return structured pass/fail flags with a conclusion string.'
---

`evaluate_stability(diff_hom_stab, c_criterion, c_expanded)` compares the absolute difference between the stability and homogeneity overall means against the stability criterion and, optionally, the expanded criterion. It returns structured pass/fail flags and a formatted conclusion string.

## Parameters

<ParamField path="diff_hom_stab" type="numeric" required>
  Absolute difference between the stability overall mean and the homogeneity overall mean. Retrieve this from the `diff_hom_stab` field in the output of `calculate_stability_stats`.
</ParamField>

<ParamField path="c_criterion" type="numeric" required>
  Basic stability criterion. Compute this with `calculate_stability_criterion`.
</ParamField>

<ParamField path="c_expanded" type="numeric">
  Expanded stability criterion (optional). When supplied the function also evaluates `diff_hom_stab` against this wider threshold. Compute this with `calculate_stability_criterion_expanded`. Omit or pass `NULL` to skip the expanded check.
</ParamField>

## Return value

Returns a named list with three fields.

<ResponseField name="passes_criterion" type="logical">
  `TRUE` if `diff_hom_stab <= c_criterion`, `FALSE` otherwise.
</ResponseField>

<ResponseField name="passes_expanded" type="logical | NA">
  `TRUE` if `diff_hom_stab <= c_expanded`. `FALSE` if `diff_hom_stab > c_expanded`. `NA` when `c_expanded` is not provided.
</ResponseField>

<ResponseField name="conclusion" type="character">
  One or two lines of human-readable text. Each line follows the pattern:

  - Pass: `"diff (0.0210) <= criterion (0.0450): MEETS STABILITY CRITERION"`
  - Fail: `"diff (0.0512) > criterion (0.0450): DOES NOT MEET STABILITY CRITERION"`

  When `c_expanded` is provided a second line is appended describing the expanded result. The two lines are joined with `\n`.
</ResponseField>

## Notes

<Note>
  The function uses a less-than-or-equal comparison. Values at exactly the criterion boundary are considered passing.
</Note>

## Example

```r
set.seed(42)

# Full stability workflow
hom_data <- matrix(rnorm(20, mean = 10, sd = 0.5), nrow = 10, ncol = 2)
hom      <- calculate_homogeneity_stats(hom_data)

stab_data <- matrix(rnorm(16, mean = 10.05, sd = 0.5), nrow = 8, ncol = 2)
stab      <- calculate_stability_stats(
  stab_sample_data       = stab_data,
  hom_general_mean_homog = hom$general_mean_homog,
  hom_stab_x_pt          = hom$x_pt,
  hom_stab_sigma_pt      = hom$MADe
)

u_hom_mean  <- hom$sw  / sqrt(hom$g  * hom$m)
u_stab_mean <- stab$sw / sqrt(stab$g * stab$m)

c_stab     <- calculate_stability_criterion(stab$hom_stab_sigma_pt)
c_stab_exp <- calculate_stability_criterion_expanded(c_stab, u_hom_mean, u_stab_mean)

result <- evaluate_stability(
  diff_hom_stab = stab$diff_hom_stab,
  c_criterion   = c_stab,
  c_expanded    = c_stab_exp
)

cat(result$conclusion, "\n")
cat("Passes basic criterion:   ", result$passes_criterion, "\n")
cat("Passes expanded criterion:", result$passes_expanded, "\n")
```
