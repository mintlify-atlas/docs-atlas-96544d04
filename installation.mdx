ptcalc is distributed as a local R package and installed using the [devtools](https://devtools.r-lib.org/) package. This page explains how to load it during development, how to install it for regular use, and how to confirm the installation is complete.

## Prerequisites

You need R (≥ 4.0 recommended) and the devtools package installed. If you do not have devtools yet, install it from CRAN:

```r
install.packages("devtools")
```

You also need a local copy of the ptcalc source directory. The examples below assume the source is at a path named `"ptcalc"` relative to your working directory. Adjust the path to match where you have cloned or extracted the package.

## Load for development

If you are actively editing ptcalc source files or exploring the code, use `devtools::load_all()`. This loads all exported and internal functions directly into your session without creating an installed copy, so changes to source files take effect after the next `load_all()` call.

```r
devtools::load_all("ptcalc")
```

<Tip>
  `load_all()` is the fastest iteration loop when modifying package internals. You do not need to reinstall after every change — just call `load_all()` again.
</Tip>

## Install as a package

For regular use where you want ptcalc available in any R session via `library()`, run a full install:

```r
devtools::install("ptcalc")
```

devtools compiles the package, registers it in your local R library, and makes it loadable by name from any working directory. You only need to repeat this step when you want to update to a newer version of the source.

## Load the installed package

After installation, load ptcalc the same way you load any R package:

```r
library(ptcalc)
```

If the package loaded successfully, R returns silently with no error message.

## Verify the installation

Confirm that all functions are available by listing the package exports:

```r
library(ptcalc)

# List all exported names
ls("package:ptcalc")
```

You should see all 21 exported functions:

```r

```

<Note>
  If `ls("package:ptcalc")` returns an error saying the package is not found, the install did not complete successfully. Check that devtools printed no errors during the `install("ptcalc")` step and that the source path is correct.
</Note>
