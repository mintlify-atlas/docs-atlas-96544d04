ptcalc is an R package that implements the statistical methods required for proficiency testing (PT) schemes under ISO 13528:2022 and ISO 17043:2024. It provides robust estimators, four score types (z, z′, zeta, En), and complete homogeneity and stability assessment workflows — all as pure functions with no external dependencies beyond `stats` and `dplyr`.

## Get started in three steps

<Steps>
  <Step title="Install ptcalc">
    Load the package from your local source directory using devtools:

    ```r
    devtools::install("ptcalc")
    library(ptcalc)
    ```
  </Step>
  <Step title="Compute robust statistics">
    Run Algorithm A on participant results to get an outlier-resistant assigned value and standard deviation:

    ```r
    values <- c(10.1, 10.2, 9.9, 10.0, 10.3, 50.0)
    result <- run_algorithm_a(values)
    cat("Assigned value:", result$assigned_value, "\n")
    cat("Robust SD:", result$robust_sd, "\n")
    ```
  </Step>
  <Step title="Evaluate participant scores">
    Calculate and evaluate a z-score for any participant result:

    ```r
    z <- calculate_z_score(x = 10.5, x_pt = result$assigned_value, sigma_pt = result$robust_sd)
    cat("z =", z, "->", evaluate_z_score(z))
    ```
  </Step>
</Steps>

<Note>
  ptcalc follows ISO 13528:2022 Annex C.3 for Algorithm A and Sections 10.2-10.5 for score calculations. All functions are pure — they have no side effects and return `NA` on invalid input rather than throwing errors.
</Note>
